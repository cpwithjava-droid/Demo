# Demo CI/CD App

A simple Spring Boot application demonstrating:

- Jenkins CI/CD
- SonarQube analysis
- Gitleaks secret scanning
- Trivy scanning
- UAT and PROD deployment
- ZAP DAST security test

Run locally:

```
mvn spring-boot:run
```

Endpoints:

- http://localhost:8080/hello
- http://localhost:8080/health
